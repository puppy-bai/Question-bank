# 题库练习平台 Web/H5 版

这是题库项目的网站版本，保留小程序的核心业务，并增加了适合 Web 运营的后台、授权和部署结构。

## 当前功能

- 用户登录 / 注册：姓名 + 手机号。
- 用户端：练习、题库、我的。
- 练习模式：顺序练习、随机练习、专项练习、章节练习、模拟考试、错题、收藏。
- 即时判题：单选题和判断题选择后立刻反馈，多选题需要点“确认答案”。
- 状态反馈：答对显示绿色，答错显示红色，并同步答题卡。
- 考试模式：整套试卷完成后统一提交批卷，满分 100 分。
- 题库中心：管理员发布题库，用户加入“我的题库”后才能练习。
- 管理后台：题库管理、Word/TXT 导入、考试模板、激活码、用户授权、订单、备份、数据统计。
- 收费预留：支持付费题库、套餐、激活码、预留订单和手动授权。

当前版本使用浏览器 `localStorage` 保存数据，适合免费静态部署、演示、试用和业务流程验证。正式收费运营前，需要把 `src/store.js` 替换为真实后端数据接口。

## 本地运行

```bash
cd web
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:5173
```

生产构建：

```bash
npm run build
```

构建产物在：

```text
web/dist
```

## 默认账号

- 用户端：输入任意姓名和手机号即可登录或注册。
- 管理员端：默认密码 `admin123`。

正式上线后，管理员密码应迁移到后端权限系统，不要继续放在前端代码里。

## 导入格式

后台导入支持 `.docx`、`.txt`、`.md`、`.csv`。静态版会从 Word 段落中提取文本，再按题目格式解析。

推荐格式：

```text
第一章 基础知识
1. 电流的单位是？（单选）
A. 伏特
B. 安培
C. 欧姆
D. 瓦特
答案：B
解析：安培是电流的国际单位。

2. 下列属于常见电路参数的是？（多选）
A. 电压
B. 电流
C. 电阻
D. 湿度
答案：A、B、C
解析：电压、电流、电阻都是常见电路参数。

3. 发现有人触电时，应先切断电源再施救。（判断）
答案：正确
解析：先切断电源可以避免二次伤害。
```

## 免费部署

推荐先部署到 Cloudflare Pages 或 Vercel。

Cloudflare Pages：

1. 把项目推送到 GitHub。
2. 打开 Cloudflare Pages，创建项目并连接 GitHub 仓库。
3. Root directory 填 `web`。
4. Build command 填 `npm run build`。
5. Build output directory 填 `dist`。
6. 部署完成后会得到 `https://xxx.pages.dev`。

Vercel：

1. Import GitHub Project。
2. Root Directory 选择 `web`。
3. Framework 选择 Vite。
4. Build Command 填 `npm run build`。
5. Output Directory 填 `dist`。

## 正式运营下一步

静态版的数据只保存在当前浏览器，不适合真实多用户共享和真实支付回调。正式收费前建议接入：

- Supabase 或 Cloudflare D1：保存用户、题库、做题记录、错题、收藏、订单、授权。
- 后端 Word 解析服务：稳定处理大文件和复杂 Word 模板。
- 支付或激活码核销：先用激活码模式最快，后续再接微信支付、支付宝或收款码审核流程。
- 用户协议、隐私政策、退款规则、客服方式和备案方案。

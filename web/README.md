# 题库练习平台 Web/H5 版

这个目录是当前微信小程序的 Web/H5 版本骨架，目标是保留核心业务流程，并为后续收费能力预留接口。

## 已包含

- 用户登录/注册
- 管理员登录
- 题库市场、加入我的题库
- 顺序练习、随机练习、专项练习、章节练习
- 模拟考试，支持默认模板和自定义题型/章节比例
- 即时判题、错题、收藏、答题卡
- 管理员题库管理、演示导入、考试模板、数据统计
- 支付入口预留

当前版本使用浏览器 `localStorage` 保存演示数据，适合免费静态部署和功能演示。正式运营时建议把 `src/store.js` 替换为真实后端适配器。

## 本地运行

```bash
npm install
npm run dev
```

## 免费部署方式

推荐先用 Cloudflare Pages 或 Vercel：

1. 把项目上传到 GitHub。
2. 在 Cloudflare Pages / Vercel 新建项目，选择 `web` 目录。
3. Build command 填：

```bash
npm run build
```

4. Output directory 填：

```bash
dist
```

5. 部署完成后会得到一个免费网址。

## 正式收费前建议补齐

- 后端数据库：Supabase 免费版、Cloudflare D1、腾讯云开发 HTTP API 或自有服务器
- 登录鉴权：手机号验证码或账号密码
- 订单系统：订单号、金额、支付状态、退款状态
- 权限系统：题库购买、会员有效期、激活码
- 支付接口：微信支付、支付宝、激活码核销或人工收款审核
- 合规文件：用户协议、隐私政策、退款规则、ICP备案

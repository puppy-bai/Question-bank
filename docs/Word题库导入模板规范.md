# Word 题库导入模板规范

版本：v0.1  
日期：2026-05-25  
适用范围：第一版优先支持 `.docx`，暂不支持老式 `.doc`  
参考文件：`C:\Users\Puppy\Desktop\人工智能材料\shuashuati_word.docx`

## 1. 设计目标

题库导入需要稳定、可校验、可预览。第一版采用参考文件中的“刷刷题风格”自由文本 Word 模板，管理员上传后由云函数解析，生成题目预览和错误报告，管理员确认后再写入正式题库。

## 2. 模板整体结构

参考文件不是表格模板，而是按段落顺序组织的文本模板：

```text
第一章/第一节
选择题格式
题干
A.选项 A
B.选项 B
C.选项 C
D.选项 D
答案：A
解析：这里填写解析

简答题格式
题干
答案：参考答案
解析：这里填写解析
```

解析规则以段落为单位读取 Word 内容。空段落用于自然分隔，但不作为唯一分隔依据。

## 3. 章节格式

章节行用于建立题库目录。支持以下格式：

```text
第一章/第一节
二、第二章/第一节
三、第三章
```

解析建议：

- 包含 `章`、`节`、`/`，且不是题干、答案、解析、选项的段落，可识别为章节。
- 章节使用 `/` 拆分层级，例如 `第一章/第一节`。
- 章节前的中文序号如 `二、`、`三、` 可自动去掉。

## 4. 选择题格式

### 4.1 显式答案行

```text
刷刷题的网址是？
A.www.shuashuati.com
B.www.google.com
C.www.baidu.com
D.www.jd.com
答案：A
解析：shuashuati.com 是刷刷题的网址。
```

### 4.2 题干括号答案

```text
刷刷题的网址是（A）
A.www.shuashuati.com
B.www.google.com
C.www.baidu.com
D.www.jd.com
解析：这里填写解析。
```

导入时应把题干中的 `（A）` 或 `(A)` 识别为答案，并从展示题干中移除或保留为可配置项。第一版建议移除，避免学生直接看到答案。

### 4.3 选项红色答案

参考文件中存在红色选项：

```text
刷刷题的网址是？
A.www.shuashuati.com
B.www.google.com
C.www.baidu.com
D.www.jd.com
解析：标红部分会直接被识别为答案
```

Word 解析时，如果某个选项整段或核心文字为红色，可识别为正确答案。由于红色也可能用于说明文字，第一版建议只在选项段落中识别红色答案。

### 4.4 选项内标记正确答案

```text
A.www.shuashuati.com（正确答案）
B.www.google.com
C.www.baidu.com
D.www.jd.com
```

导入时识别 `（正确答案）`、`(正确答案)`，并从选项展示文本中移除。

### 4.5 多选题

```text
以下哪些相关考试题目可以在刷刷题上找到？
A.小学教师资格证
B.四级人力资源师
C.一级建造师
D.计算机四级
答案：ABCD
解析：多选题
```

答案包含多个选项字母时识别为多选题。支持 `ABCD`、`A,B,C,D`、`A B C D`。

## 5. 简答题格式

```text
刷刷题的智能刷是什么意思？
答案：智能刷，会加入用户大数据分析，记忆曲线，自适应难度等因素。
解析：简答题的格式，常用于简答题、计算题、论述题
```

没有选项且存在较长文本答案时，识别为简答题。第一版简答题只展示参考答案，不自动判分。

## 6. 判断题格式

支持题干末尾直接标注答案：

```text
刷刷题的网址是 baidu.com（错误）
刷刷题的网址是 baidu.com（错）
刷刷题的网址是 baidu.com（×）
刷刷题是一个全品类的刷题网站(正确)
刷刷题是一个全品类的刷题网站(对)
刷刷题是一个全品类的刷题网站(√)
```

也支持显式答案行：

```text
刷刷题不仅有网站还有 app
答案：正确
```

导入时应把末尾括号中的判断答案识别出来，并从展示题干中移除。

## 7. 填空题格式

### 7.1 括号填空

```text
刷刷题是一个(全品类)的(刷题)网站
解析：填空题，() 里的为填空内容，支持多空
```

导入时识别括号中的内容为答案，展示给学生时替换为空：

```text
刷刷题是一个____的____网站
```

### 7.2 下划线填空

参考文件中存在下划线文字：

```text
刷刷题是一个全品类的刷题网站
解析：填空题，下划线里的为填空的内容
```

Word 解析时，带下划线的文字识别为填空答案。展示给学生时替换为下划线。

## 8. 名词解释格式

```text
刷刷题:刷刷题是一个 app，一键制作电子题库手机刷题
```

如果段落中只有一个冒号，冒号前为名词，冒号后为解释，可识别为名词解释题：

- 题干：解释“刷刷题”。
- 答案：刷刷题是一个 app，一键制作电子题库手机刷题。

## 9. 图片选项

参考文件中存在选项图片：

```text
A.
<图片>
B.www.google.com
```

第一版应支持提取 Word 内嵌图片并上传到云存储，选项内容保存为：

```json
{
  "key": "A",
  "text": "",
  "images": ["cloud://.../image1.png"]
}
```

如果图片提取复杂，可以先在导入预览中提示“检测到图片选项，需要人工确认”。

## 10. 共用题干格式

```text
(1~2题共用题干,用于案例分析，共用题干，完型填空，医护题型)
使用刷刷题后我们可以知道
刷刷题不仅有网站还有 app
答案：正确
刷刷题可以方便导入题库然后进行刷题复习(正确)
```

解析建议：

- 识别 `(1~2题共用题干...)`、`（1-2题共用题干...）` 为共用题干块开始。
- 后续题目挂载同一段材料，直到题号范围结束或遇到新章节。
- 第一版可以先保存为 `groupStem` 字段，并在答题页展示在题干上方。

## 11. 共用备选答案格式

```text
（1-3题共用备选答案，常用于医护题型)
A、原发型肺结核
B、亚急性血行播散型肺结核
C、浸润型肺结核
D、慢性纤维空洞型肺结核
E、结核性胸膜炎
1.最常见的成人继发性肺结核
答案：C
2.易导致肺气肿或肺心病的是
答案：D
3. X 线胸片可无肺内病灶的是
答案：E
```

解析建议：

- 识别 `共用备选答案` 为共享选项块开始。
- 后续 `A、B、C、D、E、` 行作为共享选项。
- 后续带数字题号的段落作为子题。
- 子题通过答案行选择共享选项。

## 12. 标题说明行

参考文件包含以下说明性标题：

```text
选择题格式
简答题格式
判断题格式
填空题格式
名词解释格式
案例分析，共用题干，完型填空，医护题型格式
```

这些段落用于提示模板格式，不应导入为题目。

## 13. 导入流程

```text
管理员进入隐藏管理入口
  -> 上传 .docx 文件
  -> 文件保存到云存储
  -> 云函数按段落和富文本标记解析 Word
  -> 生成 import_jobs 记录
  -> 展示导入预览与错误列表
  -> 管理员确认
  -> 写入 question_banks / chapters / questions
  -> 生成导入日志
```

## 14. 校验规则

- 文件必须是 `.docx`。
- 至少识别出一个章节或题目。
- 选择题至少需要 A、B 两个选项。
- 选择题必须能识别出答案，来源可以是答案行、题干括号、红色选项或正确答案标记。
- 多选题答案可以有多个选项，支持 `ABCD`、`A,B,C,D`、`A B C D`。
- 判断题答案只能是 `正确`、`错误`、`对`、`错`、`√`、`×`。
- 填空题至少需要识别出一个填空答案。
- 简答题必须存在答案行。
- 检测到图片时必须成功提取图片，或在导入预览中标记为需要人工确认。
- 无法归类的段落不直接丢弃，应进入导入报告的警告列表。

## 15. 导入报告

每次导入生成一份报告：

- 文件名。
- 上传人。
- 上传时间。
- 总行数。
- 成功解析题数。
- 错误题数。
- 错误行号和原因。
- 最终导入状态：待确认、已导入、已取消、导入失败。

```json
  "sourceParagraphs": [3, 4, 5],

```

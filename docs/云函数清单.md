# 云函数清单

版本：v0.1  
日期：2026-05-25

## 已实现云函数

| 云函数 | 用途 | 当前状态 |
| --- | --- | --- |
| `login` | 获取 OpenID，初始化普通用户 | 已实现 |
| `adminLogin` | 管理员密码登录，首次绑定管理员 OpenID | 已实现 |
| `parseWord` | 解析 `.docx` 题库并生成预览 | 已实现 |
| `confirmImport` | 确认导入预览，写入题库/章节/题目 | 已实现 |
| `getBanks` | 获取已发布题库列表 | 已实现 |
| `getQuestions` | 获取某题库题目，前端不返回答案 | 已实现 |
| `submitAnswer` | 提交答案、判题、写答题记录和错题 | 已实现 |
| `getWrongQuestions` | 获取当前学生未掌握错题 | 已实现 |
| `markWrongQuestionMastered` | 标记错题已掌握并同步未掌握错题数 | 已实现 |
| `getFavorites` | 获取当前学生收藏题目 | 已实现 |
| `toggleFavorite` | 收藏或取消收藏题目 | 已实现 |
| `teacherGetDashboard` | 老师查询负责班级练习概览 | 已实现 |
| `teacherGetStudentStats` | 老师查询单个学生答题详情 | 已实现 |
| `adminOrgManage` | 管理员维护班级、账号角色、学生分班、老师负责班级 | 已实现 |
| `adminBankManage` | 管理员维护题库可见范围 | 已实现 |
| `adminQuestionManage` | 管理员维护题目内容、答案、解析和上下架状态 | 已实现 |

# 新版功能实施清单

日期：2026-06-04

本清单用于把新版第三方题库小程序设计拆成可执行开发任务。主设计以 `docs/项目设计书.md` 为准。

## 第 1 阶段：入口与角色重构

- [ ] 新增 `pages/login/login` 作为首屏登录选择页。
- [ ] 调整 `app.json`，默认首页改为登录选择页。
- [ ] 小程序标题从旧名称改为通用题库名称。
- [ ] 用户登录后跳转用户首页。
- [ ] 管理员登录后跳转管理后台首页。
- [ ] 普通用户角色统一为 `user`。
- [ ] 移除用户端旧场景文案。
- [ ] 保留管理员密码 + OpenID 双校验。

## 第 2 阶段：用户端题库体验

- [ ] 改造首页为用户首页：继续练习、最近题库、错题数量、个人概览。
- [ ] 改造题库列表：展示已发布题库、题量、章节数、进度、正确率。
- [ ] 新增题库详情页：题库信息、章节列表、模式入口。
- [ ] 章节支持选择单章节或多章节。
- [ ] `getQuestions` 支持按 `chapterIds` 过滤。
- [ ] 练习页支持从题库、章节、错题、收藏进入。
- [ ] 答题页保持选择题和判断题点击选项作答。

## 第 3 阶段：测试模式

- [ ] 新增测试开始页：选择题库、章节、题量、是否随机。
- [ ] 复用答题页，增加 `mode=test`。
- [ ] 测试过程中不立即展示答案。
- [ ] 统一提交后生成测试结果。
- [ ] 错题写入错题本。
- [ ] 结果页展示正确率、用时、错题入口。

## 第 4 阶段：考试模式

- [ ] 新增 `exam_configs` 集合。
- [ ] 新增管理员考试配置页面。
- [ ] 新增用户考试列表或题库详情内考试入口。
- [ ] 考试答题支持倒计时。
- [ ] 到时间自动交卷。
- [ ] 新增 `exam_records` 集合保存成绩。
- [ ] 结果页展示得分、通过状态、用时和错题。

## 第 5 阶段：管理员后台

- [ ] 新增管理后台首页。
- [ ] 题库管理保留新增、重命名、发布、下架、删除、重新统计。
- [ ] Word 导入保留分批导入和解析预览。
- [ ] 新增章节管理：新增、重命名、排序、移动题目。
- [ ] 题目管理支持按题库、章节、题型、状态筛选。
- [ ] 新增用户管理：用户列表、备注名、状态、详情。
- [ ] 新增数据统计页面。

## 第 6 阶段：统计与数据

- [ ] 新增 `practice_sessions` 集合，记录每次练习/测试会话。
- [ ] 答题提交继续写入 `attempts`。
- [ ] 练习完成时聚合写入会话结果。
- [ ] 管理后台统计用户活跃、题库使用、题目错误率。
- [ ] 考试统计读取 `exam_records`。
- [ ] 高频错题按题目聚合。

## 第 7 阶段：测试验收

- [ ] 新用户首次登录。
- [ ] 管理员首次初始化。
- [ ] 管理员导入 Word 并发布题库。
- [ ] 用户能看到发布后的题库。
- [ ] 用户能选择章节练习。
- [ ] 判断题可点击正确/错误。
- [ ] 单选/多选可点击选项。
- [ ] 测试模式统一交卷。
- [ ] 考试模式倒计时交卷。
- [ ] 错题和收藏可保存。
- [ ] 管理员可删除题库。
- [ ] 用户端不出现管理功能。
- [ ] 管理员端统计数据可打开。

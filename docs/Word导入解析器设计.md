# Word 导入解析器设计

版本：v0.1  
日期：2026-05-25  
参考模板：`C:\Users\Puppy\Desktop\人工智能材料\shuashuati_word.docx`

## 1. 输入与输出

### 1.1 输入

- `.docx` 文件。
- 当前管理员 OpenID。
- 目标题库 ID，或导入时创建新题库。

### 1.2 输出

- `import_jobs` 导入任务。
- 题目预览列表。
- 错误和警告列表。
- 提取后的图片资源。

## 2. 解析步骤

```text
读取 docx
  -> 提取段落文本、富文本样式、图片关系
  -> 标准化中文/英文标点
  -> 识别章节和说明行
  -> 按状态机识别题目块
  -> 推断题型
  -> 提取答案和解析
  -> 生成题目预览
  -> 输出错误/警告
```

## 3. 段落结构

每个段落解析为：

```json
{
  "index": 12,
  "text": "A.www.shuashuati.com",
  "runs": [
    {
      "text": "A.www.shuashuati.com",
      "color": "FF0000",
      "underline": false,
      "bold": false,
      "imageIds": []
    }
  ],
  "images": []
}
```

## 4. 识别规则

### 4.1 章节

匹配：

- `第一章/第一节`
- `二、第二章/第一节`
- `三、第三章`

识别后更新 `currentChapter`。

### 4.2 说明行

以下段落跳过：

- `选择题格式`
- `简答题格式`
- `判断题格式`
- `填空题格式`
- `名词解释格式`
- 包含 `题型格式` 且明显是模板说明的行

### 4.3 选项

支持：

- `A.xxx`
- `A. xxx`
- `A、xxx`
- `A、 xxx`

选项可以包含图片。

### 4.4 答案

支持：

- `答案：A`
- `答案: A`
- `答案：ABCD`
- 题干末尾 `(A)`、`（A）`
- 选项中的 `（正确答案）`
- 选项段落中的红色文字
- 判断题末尾 `(正确)`、`（错）`、`(√)`、`（×）`
- 填空题括号内容和下划线文字

### 4.5 解析

支持：

- `解析：...`
- `解析: ...`

解析段落归属于上一道题。

## 5. 状态机

```text
Idle
  -> ChapterDetected
  -> QuestionStarted
  -> CollectingOptions
  -> WaitingAnswer
  -> WaitingAnalysis
  -> FinalizeQuestion
```

### 5.1 普通选择题

1. 遇到非章节、非说明、非答案、非解析、非选项段落，开启题干。
2. 后续连续选项段落进入 `CollectingOptions`。
3. 遇到答案行、解析行或下一题时完成题目。
4. 有多个答案字母则为多选，否则为单选。

### 5.2 判断题

1. 段落末尾包含判断答案，直接生成判断题。
2. 或没有选项且答案行为 `正确/错误/对/错/√/×`，生成判断题。

### 5.3 填空题

1. 段落中有括号答案且不是选项题答案，生成填空题。
2. 段落 run 中有下划线文字，生成填空题。
3. 展示题干中把答案替换为 `____`。

### 5.4 简答题

1. 没有选项。
2. 存在答案行。
3. 答案不是选项字母，也不是判断值。
4. 生成简答题。

### 5.5 共用题干

1. 遇到 `共用题干` 行，进入共享材料模式。
2. 后续材料段落保存为 `groupStem`。
3. 指定题号范围内的题目挂载该 `groupStem`。

### 5.6 共用备选答案

1. 遇到 `共用备选答案` 行，进入共享选项模式。
2. 后续 A-E 选项保存为 `sharedOptions`。
3. 带数字题号的段落作为子题。
4. 子题使用共享选项和自己的答案行。

## 6. 错误分级

### 6.1 错误

错误会阻止单题导入：

- 选择题缺少答案。
- 选择题少于两个选项。
- 答案字母不在选项中。
- 判断题答案无法识别。
- 填空题没有识别到填空答案。

### 6.2 警告

警告不阻止导入，但需要在预览中提示：

- 检测到图片。
- 检测到红色文字但不在选项中。
- 段落无法归类。
- 题目没有解析。
- 未识别章节，归入默认章节。

## 7. 第一版实现边界

第一版必须支持：

- 章节。
- 单选、多选。
- 判断。
- 填空括号答案。
- 简答。
- 答案行和解析行。
- 题干括号答案。
- 选项正确答案标记。
- 红色选项答案。

第一版可降级支持：

- 图片选项：可先提取并预览，导入前提示人工确认。
- 下划线填空：优先支持，若解析库获取样式不稳定则给警告。
- 共用题干和共用备选答案：先按参考文件支持基础场景。

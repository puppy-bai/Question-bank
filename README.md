## 当前默认方向

- 客户端：微信原生小程序
- 后端：微信云开发 / 腾讯云 CloudBase
- 第一版用户：学校内部学生、老师、管理员
- 第一版目标：按固定 Word 模板导入题库、按章节/分类刷题、自动判题、错题收藏、练习记录、老师查看学生答题数据、隐藏管理入口
- 学生学习能力：答题、背题、错题本复习、标记已掌握、收藏夹、本地断点续答和练习结果页
- 第一版管理能力：管理员可在隐藏入口维护 Word 导入、题目内容、账号角色、班级、学生分班、老师负责班级和题库可见范围
- 管理方式：小程序内隐藏管理入口，管理员密码 + OpenID 绑定
- 商业化：第一版不做收费和支付
